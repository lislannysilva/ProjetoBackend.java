# Saúde Conectada - App de Saúde Preventiva e Informativa

## 📋 Descrição do Projeto
O **Saúde Conectada** é uma plataforma desenvolvida para facilitar o acesso da população aos serviços básicos de saúde. O foco principal é a medicina preventiva, permitindo que cidadãos gerenciem sua saúde de forma digital e recebam informações cruciais em tempo real.

## 🎯 Objetivo
Reduzir as filas presenciais em unidades de saúde e aumentar a cobertura vacinal através de:
- Democratização do agendamento de consultas.
- Disseminação de campanhas de saúde pública.
- Orientação rápida sobre primeiros socorros e bem-estar mental.

## 🛠 Tecnologias Usadas
- **Frontend:** Flutter (Dart)
- **Backend:** Python / Java (API REST)
- **Banco de Dados:** Oracle SQL / PL-SQL
- **Documentação:** Swagger / Postman

## 🚀 Como rodar o projeto
1. Clone o repositório: `git clone [URL_DO_REPO]`
2. Instale as dependências: `flutter pub get` (ou comando equivalente da stack)
3. Configure as variáveis de ambiente no arquivo `.env`.
4. Execute o projeto: `flutter run`.
